# Network Security

# Introduction

Network security focuses on protecting computer networks and connected systems from unauthorized access, attacks, and disruptions.

It is a critical part of modern cybersecurity.

---

# Goals of Network Security

- Protect data
- Prevent unauthorized access
- Monitor network traffic
- Detect attacks
- Maintain service availability

---

# Common Network Components

| Component | Purpose |
|---|---|
| Router | Direct network traffic |
| Switch | Connect devices |
| Firewall | Filter traffic |
| IDS/IPS | Detect intrusions |
| VPN | Secure communication |

---

# Firewalls

Firewalls monitor incoming and outgoing traffic.

They can:
- allow safe traffic
- block suspicious traffic

Types:
- hardware firewalls
- software firewalls

---

# Intrusion Detection Systems

IDS systems monitor networks for suspicious activity.

Examples:
- unusual traffic
- attack signatures
- unauthorized access attempts

---

# VPN Security

VPNs encrypt internet traffic.

Benefits:
- secure remote access
- IP masking
- safer public Wi-Fi usage

---

# Common Network Attacks

# DDoS Attacks

Attackers overwhelm systems with excessive traffic.

---

# Man-in-the-Middle Attacks

Attackers intercept communication between parties.

---

# Packet Sniffing

Capturing and analyzing network packets.

---

# Spoofing

Forging network identities or addresses.

---

# Security Best Practices

# Use Strong Passwords

Secure network devices and accounts.

---

# Update Firmware

Install security patches regularly.

---

# Segment Networks

Separate sensitive systems from public networks.

---

# Monitor Traffic

Identify suspicious activity early.

---

# Encrypt Communications

Use HTTPS, VPNs, and secure protocols.

---

# Wireless Network Security

Secure Wi-Fi using:
- WPA2
- WPA3
- strong passwords

Avoid open public networks when possible.

---

# Zero Trust Security

Zero Trust assumes no user or device is automatically trusted.

Verification is required continuously.

---

# Importance of Monitoring

Continuous monitoring helps:
- detect attacks
- identify anomalies
- improve response times

---

# Conclusion

Network security is essential for protecting systems, data, and communication infrastructure from modern cyber threats.
