# Security Policy

# Reporting Security Issues

If you discover security-related problems in repository code or documentation:

- Do not publicly disclose sensitive issues immediately
- Report responsibly through repository communication channels

---

# Scope

This repository is educational and research-focused.

No malicious tools or illegal activities are supported.

---

# Supported Content

The repository supports:

- educational cybersecurity
- privacy research
- documentation
- legal research material

---

# Responsible Disclosure

Contributors should practice responsible disclosure when discussing vulnerabilities or security concepts.

---

# Safety Guidelines

Users should:
- test only authorized systems
- follow laws and regulations
- use cybersecurity knowledge ethically
